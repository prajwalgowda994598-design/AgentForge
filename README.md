AgentForge 🚀

Autonomous Multi-Agent AI Research System

AgentForge is a production-grade autonomous multi-agent AI research platform that orchestrates specialized AI agents to perform deep research, fact-checking, reasoning, and report generation with minimal human intervention.

🌟 Features

- Multi-Agent Architecture
  
  - Researcher Agent
  - Summarizer Agent
  - Critic Agent
  - Fact-Checker Agent
  - Synthesizer Agent

- Retrieval-Augmented Generation (RAG)

- Shared Vector Memory using FAISS

- Knowledge Graph Integration with Neo4j

- Real-Time Agent Reasoning Visualization

- WebSocket Streaming

- Self-Healing Agent Workflows

- AWS Cloud Deployment

- FastAPI Backend

- Docker Containerization

---

🚀 Getting Started

Clone Repository

git clone https://github.com/yourusername/AgentForge.git
cd AgentForge

Install Dependencies

pip install -r requirements.txt

Run Backend

uvicorn app.main:app --reload

Run Frontend

npm install
npm run dev
